# Nv-engine-v2

---
Nv作为二代引擎，是Zr引擎的重构版本。
优化和修复功能如下：
 > 加载不流畅

### 使用技巧

代码clone下来后，通过yarn方式安装，如果没有安装，请按照[官网流程安装](https://www.yarnpkg.com/zh-Hans/)

安装步骤如下：

```node
$ clone git https://github.com/guguaihaha/Zr-engine-v2.git

$ yarn install

//启动开发环境并同时启动热加载
$ npm run dev

//生成上线压缩并优化的代码
$ npm run build
```
